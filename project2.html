Australian Vehicle Price Prediction

Overview

This project predicts vehicle prices using a machine learning pipeline. It includes data preprocessing, model training, and a Streamlit app for real-time predictions.

Technologies Used

  • Python
  • Pandas
  • Scikit-learn
  • Streamlit
  • Matplotlib

Key Features

  • Real-time prediction deployment
  • Support for multiple vehicle attributes
  • Interactive user interface using Streamlit
